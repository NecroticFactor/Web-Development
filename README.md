
# CS50W Project Repository

Welcome to the repository for all the projects completed as part of the **CS50W: Web Programming with Python and JavaScript** course offered by Harvard University. This repository contains the code, files, and documentation for each project assigned throughout the course.

## Projects Overview

The course covers various web development concepts, such as HTML, CSS, JavaScript, Django, SQL, and more. Below is a brief summary of the projects in this repository:

1. **Project 0: Search**  
   Replicating Google's Advanced Search page using HTML and CSS.
   
2. **Project 1: Wiki**  
   Building a Wikipedia-like website using Django.
   
3. **Project 2: Commerce**  
   Creating an e-commerce auction site where users can list, bid, and comment on items.
   
4. **Project 3: Mail**  
   Developing a front-end for an email client using JavaScript to send and receive emails.
   
5. **Project 4: Network**  
   Implementing a social media platform where users can post and follow others.
   
6. **Project 5: Capstone**  
   A final project that showcases all the skills learned during the course.

## Requirements

Each project folder includes a `requirements.txt` file listing the necessary Python packages, ensuring the environment can be set up quickly.

To install dependencies, run the following command:

```bash
pip install -r requirements.txt
```

## How to Run

Navigate into each project's directory and follow the instructions in the respective `README.md` or `instructions.txt` files to run the application locally.
